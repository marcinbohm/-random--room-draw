# -random--room-draw
Room draw but on third click you get number you wanted before

Program made in HTML with Javascript

I had funny situation few days ago. My friend told me he is going on holidays with his friends from school. 
They booked a hotel with 8 rooms. But the best room is with number 3. 
So he need room draw program where on third click always will be room number 3. 
He will draw as the third. Other clicks are random. Check this out !!!
